# Contributing to Focus Timer

Thanks for your interest in improving Focus Timer! 🎉

## How to contribute
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-idea`
3. Make your change and test it by opening `index.html` in a browser
4. Commit with a clear message
5. Open a pull request describing what you changed and why

## Ideas welcome
- New themes / color schemes
- Adjustable work and break lengths
- A sound or notification when a session ends

Please keep the project dependency-free — it's intentionally a single set of static files.
